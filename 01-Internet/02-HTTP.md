# ¿Que es un protocolo?

- Es conjunto de reglas y estandares usadas para la comunicacion entre maquinas.

# ¿Que es HTTP?

- HyperText Transfer Protocol o HTTP es un protocolo de comunicacion para transmitir documentos como HTML.

### ¿HTTPS?

- La informacion que viaja a traves de HTTP puede ser interceptada por terceros y ser usada para otros fines, para solucionar ese problema la informacion viaja encriptada.

- El protocolo se cifra mediante TLS (Transport Layer Security) o mediante su antecesor SSL (Secure Sockets Layer).

- Mediante el certificado SSL, el servidor y el navegador manejan una conexion segura y cifrada.
