# ¿Qué es una IP?

- Los dispositivos tienen una dirección única, esta dirección es un número.

- IP (Internet Protocol) o protocolo de internet usa esta dirección única llamada IP address o dirección IP. La dirección IP es usada para la comunicación de dispositivos a través de internet.

- Cuando un dispositivo solicita recursos a otro, envía su propia dirección IP en la solicitud para que el receptor sepa a donde enviar la información.

### Composición de una dirección IP

- Las IP tradicionales (IPv4) se componen de 4 números del 0 al 255 separados por puntos, se pueden representar con 4 octetos de bits para un total de 32 bits. Ejemplo:

  - 00000000 . 00000000 . 00000000 . 00000000

- En la actualidad se está migrando a IPv6, que se componen de 8 cuartetos de números hexadecimales, se pueden representar con 16 bits por cada cuarteto para un total de 128 bits. Ejemplo:

  - 0000000000000000 : 0000000000000000 : 0000000000000000 : 0000000000000000 : 0000000000000000 : 0000000000000000 : 0000000000000000 : 0000000000000000

# ¿Qué es Domain Name?

- El nombre de dominio corresponde a la dirección IP de un sitio web, este se compone de un nombre y una extensión.

- Este sirve para encontrar fácilmente el sitio sin necesidad de memorizar la dirección IP.
