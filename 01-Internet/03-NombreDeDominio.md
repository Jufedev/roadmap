# ¿Que es una IP?

- Los dispositivos tienen una direccion unica, esta direccion es un numero.

- IP (Internet Protocol) o protocolo de internet usa esta direccion unica llamada IP address o direccion IP. La direccion IP es usada para la comunicacion de dispositivos a traves de internet.

- Cuando un dispositivo solicita recursos a otro, envia su propia direccion IP en la solicitud para que el receptor sepa a donde enviar la informacion.

### Composicion de una direccion IP

- Las IP tradicionales (IPv4) se componen de 4 numeros del 0 al 255 separados por puntos, se pueden representar con 4 octetos de bits para un total de 32 bits. Ejemplo:

  - 00000000 . 00000000 . 00000000 . 00000000

- En la actualidad se esta migrando a IPv6, que se componen de 8 cuartetos de numeros hexadecimales, se pueden representar con 16 bits por cada cuarteto para un total de 128 bits. Ejemplo:

  - 0000000000000000 : 0000000000000000 : 0000000000000000 : 0000000000000000 : 0000000000000000 : 0000000000000000 : 0000000000000000 : 0000000000000000

# ¿Que es Domain Name?

- El nombre de dominio corresponde a la direccion IP de un sitio web, este se compone de un nombre y una extension.

- Este sirve para encontrar facilmente el sitio sin necesidad de memorizar la direccion IP.
