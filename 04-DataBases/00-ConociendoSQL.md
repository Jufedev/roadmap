# ¿Que es una base de datos?

- Es un conjunto de datos estructurados que pertenecen a un mismo contexto.

- Se usa para administrar la informacion.

# ¿Que es SQL?

- Structured Query Language o SQL es un lenguaje de consulta diseñado para administrar y recuperar informacion de sistemas de gestion de bases de datos relacionales.

- Fue uno de los primeros lenguajes comerciales para el modelo relacional de Edgar Frank Codd.

- SQL paso a ser el estandar del ANSI en 1986 y de la ISO en 1987.

- Las palabras reservadas se escriben en mayuscula (SELECT, FROM, etc).
