- **Archivos MDF:**

  - Master Data File es el principal archivo de datos.

  - Contiene las tablas, indices, procedimientos almacenados y otros objetos de la base de datos.

- **Archivos LDF:**

  - Log Data File es el archivo de registro de transacciones.

  - Registra todas las transacciones y cambios realizados en la base de datos, lo que permite la recuperacion y la restauracion de la misma en caso de fallos o perdida de datos.
