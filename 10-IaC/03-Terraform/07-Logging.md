# Logs

- Se usan al hacer un plan o apply.

- Debuggear problemas.

- Niveles de detalle:
    - Info  (Menor nivel de logs)
    - Warning
    - Error
    - Debug
    - Trace (Muestra todo)

- Se configura con la variable de entorno `TF_LOG=VALOR`.

- Se puede usar un archivo para guardar log, se configura con la variable `TF_LOG_PATH=terra_log.txt`.
